# Adonai Hernández Bolaños
Ingeniero especializado en sistemas embebidos, arquitectura de hardware y aceleración computacional. Enfocado en la optimización de algoritmos de bajo nivel, diseño RTL y sistemas de alto rendimiento.

## 🛠 Áreas de Especialización

* **Aceleración de Hardware:** Diseño y optimización de kernels para FPGAs utilizando OpenCL (Intel FPGA SDK).
* **Sistemas Embebidos:** Desarrollo bare-metal, diseño de sistemas basados en RTL y flujo de trabajo para FPGAs (Vivado, Xilinx Vitis).
* **Lenguajes de Programación:** C (sistemas y rendimiento), Rust (seguridad y concurrencia), Verilog (diseño de hardware), Assembly.
* **Procesamiento de Señales:** Implementación de motores de cómputo y aceleración para tareas de visión artificial.

## 🚀 Proyectos Destacados

* **Convolution Engine (Intel FPGA):** Implementación de un motor de convolución de alto rendimiento utilizando OpenCL para maximizar el throughput y la eficiencia en FPGAs de Intel.
* **GBA Development:** Desarrollo de software bare-metal para Game Boy Advance. Exploración de las limitaciones y capacidades del hardware mediante C y optimización de recursos.
* **Embedded Tooling:** Desarrollo de herramientas de apoyo para la optimización de assets (conversión y empaquetado) en entornos de memoria limitada.

## 💻 Stack Técnico

* **Hardware & FPGA:** Intel FPGA SDK for OpenCL, Xilinx Vitis, Vivado, Verilog, RTL Design.
* **Lenguajes:** C, Rust, Assembly, Verilog, C#, JavaScript.
* **Conceptos Clave:** Arquitectura de computadores, procesamiento de señales (DSP), optimización de algoritmos, sistemas de tiempo real, bare-metal.
